# Dream Transmission

Dream Transmission is a meditative ambient music player with a reactive sky scene built in React and Three.js. It combines curated audio, soft motion, and a calm visual palette.

## Highlights

- Animated star field, drifting clouds, and shooting stars
- Ambient and classical playback with persistent player state
- Multiple color themes and weather-style soundscapes
- Cloudflare-ready build and deployment flow

## Stack

- React
- Three.js
- Framer Motion
- Vite

## Quick start

```bash
bun install
bun run dev
```

Build and deploy:

```bash
bun run build
bun run deploy
```
